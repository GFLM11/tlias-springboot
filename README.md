# tlias-springboot
itheima部门管理
# 描述
基于SpringBoot框架构建工程；Mybatis持久层框架来管理dao层的数据访问；数据库服务器：MySQL
## 功能 
(1) 部门管理<br>
(2) 员工管理<br>
(3) JWT令牌登录校验<br>
(4) 阿里云OSS存储服务应用<br>
(5) 全局异常处理<br>
(6) AOP编程实现操作日志记录

## Maven工程设计思想
(1) 分模块设计<br>
(2) 父工程与子工程具有继承关系，父工程进行同一的依赖管理和版本锁定，父工程打包方式为pom<br>
(3) 父工程实现全部工程之间的聚合功能，方便项目快速构建
